## MVP 方針

仕様書には AI スコアリングが含まれているが、初期版では外部 AI を使わず、明示的なルールベースの不要度スコアとして実装する。

スコア算出式:

```text
不要度スコア = 最終開封日スコア * 0.55 + カテゴリスコア * 0.30 + 受信頻度スコア * 0.15
```

## 主要機能

- Gmail / Outlook / iCloud / Yahoo! Mail のアカウント管理
- 全アカウント横断の購読メール一覧
- スコア順、サービス別、カテゴリ別の絞り込み
- 送信元ドメインによる名寄せ表示
- 解除、保持操作
- 月次受信数、解除実績、推定削減数の分析
- スコアリング重みのカスタマイズ
